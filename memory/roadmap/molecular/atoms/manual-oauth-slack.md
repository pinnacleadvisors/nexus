# Manual — OAuth app: Slack

✅ Done. https://api.slack.com/apps → Create New App → From scratch. Redirect URI: `https://<your-vercel-domain>/api/oauth/slack/callback`. Add `SLACK_CLIENT_ID` + `SLACK_CLIENT_SECRET` to Doppler.
