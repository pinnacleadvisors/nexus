# Manual — Inngest background jobs setup

⬜ Not done. Sign up at https://inngest.com → create app `nexus`. Add `INNGEST_EVENT_KEY`, `INNGEST_SIGNING_KEY`, `NEXT_PUBLIC_APP_URL` (Vercel deployment URL) to Doppler. Register endpoint in Inngest dashboard → Syncs → Add: `https://<your-vercel-domain>/api/inngest`. Local dev: `npx inngest-cli@latest dev` alongside `npm run dev`.
