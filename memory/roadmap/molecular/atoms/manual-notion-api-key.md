# Manual — Optional NOTION_API_KEY

⬜ Not done. Optional: Set `NOTION_API_KEY` in Doppler — internal integration token (bypasses OAuth, for server-side agent writes; Notion membership required). Create at https://www.notion.so/my-integrations → New integration → copy "Internal Integration Token". Share target pages/databases with the integration.
