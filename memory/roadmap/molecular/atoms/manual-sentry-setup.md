# Manual — Sentry error tracking setup

✅ Done. Run `npm install @sentry/nextjs`, then `npx @sentry/wizard@latest -i nextjs` (generates config files). Add `SENTRY_DSN` to Doppler (Sentry project → Settings → Client Keys).
